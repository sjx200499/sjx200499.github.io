## 🎯 一句话总结 (TL;DR)
本项目针对工业污水过滤设备的透水率衰减问题，量化了自然老化、季节波动及维护操作对设备性能的影响，并通过网格搜索与蒙特卡洛模拟，制定了兼顾经济性与鲁棒性的最优维护策略，实现单台设备年均运维成本最低至 22.81 万元且永续运行。

## 📖 1. 问题背景与摘要 (Abstract)
工业污水过滤设备在使用过程中，由于污物附着与自然老化，透水率会持续下降；同时受季节温度影响，透水率呈现周期波动。
本项目旨在通过历史数据建立透水率演化模型，并寻找使得工厂污水净化年均成本最小化的中、大维护最优频率组合。

---

## ⚙️ 2. 核心数学模型 (Methodology)

### 2.1 透水率加乘混合动力模型
我们采用时间序列分解思想，完整刻画了透水率随时间、季节周期及维护操作的演变规律。透水率预测核心公式如下：

$$P(t)=[P_{0}-k\cdot t-\alpha\cdot N_{L}(t)]\cdot S(t)+\Delta_{M}\cdot N_{M}(t)+\Delta_{L}\cdot N_{L}(t)$$

> **核心参数说明：**
> * $k$: 透水率自然衰减系数 (经鲁棒回归求得)。
> * $S(t)$: 季节调节乘数 (使用三角函数拟合)。
> * $\Delta_{M}, \Delta_{L}$: 中维护与大维护的透水率单次恢复量。
> * $\alpha$: 大维护造成的不可逆折损系数。

### 2.2 修正解析寿命预测模型
引入“冬季连续30天平均透水率低于37”的工程失效准则，并定义了**有效衰减率 ($\alpha_{eff}$)**。通过扣除维护周期内的“锯齿效应”，实现了对10台设备寿命的精准预测。

## 📊 3. 优化结果与决策建议 (Results)

### 3.1 理论最优维护方案 (基于网格搜索)
以全生命周期年均总成本最小化为目标，我们在决策空间内遍历求解：
* **最优策略：** 中维护间隔设为 **48天**，**不执行大维护**。
* **优化效果：** 10台设备均可实现永续运行，彻底消除更换成本，年均成本降至 **22.81万元/台**。相比企业当前保守方案，平均每台每年节省约5万元。

### 3.2 稳健性维护方案 (基于蒙特卡洛模拟)
考虑到现实中物理参数（如衰减率）存在估计偏差，我们进行了敏感性分析与3000次蒙特卡洛联合模拟。
* **风险发现：** 理论最优方案（48天）的安全裕度极低（仅1.5%），参数微小波动即可导致永续条件破坏。
* **最终推荐方案：** 将中维护间隔缩短为 **45天**。成本仅增加 6.7%，但安全裕度提升至 8.3%，永续概率由 57.2% 飙升至 85.5%，是工程实际中抗风险能力更强的稳健选择。
